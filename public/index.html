Topology

Topology introduces RAM-like structures to the world computer1 for the first time.

Founded in late 2021, we are the protocol builders of Distributed Real-time Programs2, an open peer-to-peer Internet protocol, and Mandu3, an orchestration layer.

Combined with existing distributed hard drives4 (blockchains), the world computer can now compute in real time.
